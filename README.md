# Crucible Moments Clon

official website: [Crucible Moments](cruciblemoments.com/)

## How to install and Run

```bash
# Clone this project
git clone https://github.com/JeroPratto/crucible-moments.git
# Install
npm install
# Run
npm run dev
```

## StoryBook

```bash
# Run Storybook
npm run storybook
# Run Tests
npm run test-storybook
```

## Technologies

- React
- StoryBook
- Vite
- Typescript
